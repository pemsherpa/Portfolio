## Tech Stack

- **Backend**: Flask (Python)

- **Graphics**: p5.js for the cool particle effects
- **Icons**: Font Awesome

## How to Run It

1. **Install Python** (3.8 or higher)
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the app**:
   ```bash
   python app.py
   ```
4. **Open your browser** and go to: `http://localhost:5001`
